# vue-router

## 安装
1.cdn  
```
<script src="/path/to/vue.js"></script>
<script src="/path/to/vue-router.js"></script>
```
2.npm/cnpm  
npm/cnpm install vue-router  
## 使用
Vue.use()安装路由功能  
```
import Vue from 'vue'
import VueRouter from 'vue-router'

Vue.use(VueRouter)
```
